Partner Logo Files Required:
============================

Please add the following logo image files to this folder:

1. gsa-live.png (or .jpg, .svg) - GSA Live logo
2. riyadh-front.png (or .jpg, .svg) - Riyadh Front logo
3. national-center.png (or .jpg, .svg) - National Center for Documents and Archives logo
4. sharek.png (or .jpg, .svg) - Sharek logo
5. sfd.png (or .jpg, .svg) - Saudi Fund for Development (SFD) logo

Supported formats: PNG, JPG, JPEG, SVG
Recommended size: 200-300px width, maintain aspect ratio
Background: Transparent or white recommended

Note: If a logo file is missing, the text fallback will be displayed automatically.
